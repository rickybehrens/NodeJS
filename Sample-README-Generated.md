## Project Description
This application will generate a professional looking REAME.md main framework.

## Installation
To install necessary dependencies, run the following command:
```
npm i
```

  
## Usage
The user needs to be aware that the README file generated by this application is in no way a finished README. The user will have to update the file as the scope and the repo expand.
